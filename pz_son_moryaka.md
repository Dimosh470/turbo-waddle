## ВВЕДЕНИЕ

Выбор темы проекта ("Веб-сайт курортного отеля 'Сон Моряка'") обусловлен возрастающим спросом на качественные информационные ресурсы в сфере туризма и гостиничного бизнеса. 

В современном мире интернет стал основным средством поиска информации об отелях, размещении, услугах и ценах. Потребители ожидают от веб-сайтов удобного интерфейса, быстрой загрузки, полной информации о предоставляемых услугах и возможности онлайн-бронирования.

Создание многостраничного интернет-сайта для отеля позволит:
- Привлечь новых клиентов
- Предоставить полную информацию об услугах и номерах
- Облегчить процесс бронирования
- Улучшить имидж компании
- Обеспечить 24/7 доступность информации

Данный проект демонстрирует применение современных веб-технологий для создания профессионального информационного портала.

### 1.2 Анализ аналогичных ресурсов

Для разработки сайта отеля "Сон Моряка" был проведен анализ популярных отельных веб-сайтов и сервисов бронирования:

#### Анализируемые сайты:
1. **Booking.com** — международный сервис бронирования
   - Плюсы: удобный поиск, фильтры, отзывы, гибкая система цен
   - Минусы: перегруженный интерфейс, много рекламы

2. **Agoda.com** — азиатский сервис бронирования
   - Плюсы: мобильная оптимизация, специальные предложения
   - Минусы: сложная система фильтров

3. **Hotels.com** — сервис поиска отелей
   - Плюсы: прозрачные цены, программа лояльности
   - Минусы: не все отели имеют собственное описание

#### Выявленные тренды:
- **Фиксированная навигация** — при прокрутке страницы меню остается видимым
- **Адаптивный дизайн** — корректное отображение на мобильных устройствах
- **Быстрая загрузка** — оптимизация изображений и кода
- **Система фильтров** — удобный поиск по параметрам (цена, тип номера, дата)
- **Онлайн-калькулятор** — мгновенный расчет стоимости бронирования
- **Социальные сети** — ссылки на профили в соцсетях
- **Отзывы клиентов** — доверие новых посетителей

#### Особенности сайта "Сон Моряка":
Разработанный сайт сочетает лучшие практики анализируемых ресурсов, но с упором на:
- Элегантный и минималистичный дизайн
- Четкую структуру информации
- Интуитивную навигацию
- Удобную систему выбора номеров и услуг
- Прозрачное отображение цен

### 2.2 Цветовая гамма сайта

Цветовая гамма сайта построена на контрастном сочетании черного, белого и серых оттенков, создавая элегантный и профессиональный вид.

| Цвет | Код | Применение |
|------|-----|-----------|
| Черный | #000000 | Header, footer, кнопки, заголовки |
| Белый | #FFFFFF | Основной фон, текст, карточки |
| Темно-серый | #333333 | Основной текст |
| Средний серый | #666666 | Вспомогательный текст, границы |
| Светло-серый | #DDDDDD | Границы элементов, фон карточек |
| Очень светлый серый | #F5F5F5 | Фоновые секции |

**Принципы использования:**

- **Основной фон (#FFFFFF)**: используется как основной цвет сайта, фон карточек товаров и большинства элементов
- **Фон навигации и подвала (#000000)**: применяется для контраста и выделения ключевых элементов
- **Цвет основного текста (#333333)**: выбран для заголовков, названий и основного текста
- **Вспомогательный текст (#666666)**: используется для текста, который является описанием или второстепенной информацией
- **Цвет границ (#DDDDDD)**: используется для обрамления карточек и разделяющих элементов

Монохромная цветовая схема обеспечивает:
- Профессиональный внешний вид
- Легкость чтения текста
- Универсальность на различных устройствах
- Фокусировку внимания на содержимое

### 2.3 Описание блоков сайта

Интерфейс сайта "Сон Моряка" разделен на несколько функциональных блоков, которые повторяются на всех страницах и обеспечивают единообразность и удобство навигации.

#### JavaScript калькулятор бронирования
```javascript
function updateTotalPrice() {
    let basePrice = roomPrices[roomType] * nights;
    let discount = 1;
    
    // Применяем скидки
    if (offerFamily) discount *= 0.85; // -15%
    if (offerEarly) discount *= 0.8;   // -20%
    
    let total = basePrice * discount;
    
    // Доплата за доп. взрослых: (кол-во - 1) * 500 * ночи
    total += (adults - 1) * 500 * nights;
    
    // Доплата за детей: кол-во * 300 * ночи
    total += children * 300 * nights;
    
    // Дополнительные услуги
    if (privateBeach) total += 1500;
    if (aquapark) total += 2000;
    if (spa) total += 3000;
    
    document.getElementById('total-price').textContent = total + ' ₽';
}
```

### 2.5 Осуществление навигации по сайту

Навигация по сайту отеля "Сон Моряка" реализована с помощью взаимосвязанных элементов, обеспечивающих удобное перемещение между разделами.

#### Способы навигации:

1. **Главное навигационное меню** (header)
   - Ссылки на основные разделы
   - Остается видимым при прокрутке
   - Активная ссылка выделяется

2. **Логотип** в шапке
   - Всегда ведет на главную страницу
   - Кликабелен с любой страницы

3. **Якорные ссылки** на главной странице
   - Прямой переход к разделам (#home, #about, #rooms и т.д.)
   - Плавная прокрутка к нужной секции

4. **Кнопки действия**
   - "Забронировать" на карточках номеров
   - "Выбрать предложение" в hero секции
   - "Подробнее" на карточках услуг

5. **Карточки товаров/услуг**
   - Клик на карточку ведет на подробную страницу
   - Красивая анимация при наведении

6. **Подвал (footer)**
   - Ссылки на все основные разделы
   - Ссылка на 404 страницу (для тестирования)

7. **Параметры URL**
   - room-booking.html?room=luxury - выбор типа номера
   - Помощь в навигации между типами номеров

#### Интерактивность:

- **Hover эффекты** - карточки поднимаются при наведении
- **Плавные переходы** - transition 0.3s на всех интерактивных элементах
- **404 страница** - интерактивная с фактами о море
